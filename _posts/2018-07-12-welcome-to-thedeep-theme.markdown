This is the default theme for The Deep Jekyll sites.

How to use this theme.

update `_config.yml`:
```yml
...
remote_theme: the-deep/thedeep-theme
...
```

replace `assets/main.scss` with
```scss
---
---

@import "thedeep-theme";
```
